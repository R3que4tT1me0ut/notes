### Install
``` bash
  yum -y install wget
  wget http://dl.wdlinux.cn/lanmp_laster.tar.gz
  tar zxvf lanmp_laster.tar.gz
  sh lanmp.sh
```

### Install PHP
``` bash
wget http://down.wdlinux.cn/in/phps.sh
sh phps.sh
# php version = 5.2.17/5.3.29/5.4.45/5.5.38/5.6.30/7.0.18/7.1.4
sh phps.sh select-php-version
```
